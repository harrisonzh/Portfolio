Features and Functions

After creating the concept map, I list the features and functions of landmark ar.

  • The user can use this AR app to put a full scale building model in a big open space, and walk inside the building.
  • The user can use this AR app to put a small scale building model in front of him or her to learn about the architecture and its history. Users can zoom in and out and rotate the 3d model.
  • Users can read a short description of the landmark.
  • Users can listen to a related music to feel the environment.

Task Flow

After I finished researching, I started to work on the task flow, which was divided into 5 tasks:

  1. View full scale AR.
  2. View Miniature Model.
  3. View in AR.
  4. View introduction.
  5. Turn on/off background music.
