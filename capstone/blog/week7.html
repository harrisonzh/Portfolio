Week7 - March 12, 2021

Journal Entry 7

For this week’s progress, I work on looking for landmark architecture 3D models and edit the models.

I still spend a lot of time modifying the model, inverting the surface, making textures, and compressing the model this week . The model I edited last week is still too big. Because the model was too large so it makes Unity become lagging, and the main reason for a large model file is because of the number of faces, so I have to abandon all the Gothic buildings, including the Big Ben and Notre Dame Cathedral that I have purchased, Gothic architecture has a complex structure, and the number of faces make the model so large that it is difficult to further compress it.
I found a set of Japanese architectural models, the size is right but there are no textures. Fortunately, I found some architectural materials in Unity’s assets store, including metal, stone and wood. These materials make the building look real.
In order to reduce the model of the Chinese building, I deleted all the tiles on the roof, and made four rectangular parallelepipeds and several cylinders as the roof, which reduced the size of the model by about half. Unfortunately, most model from the "Han and Tang Dynasty Heritage" are too large to use, even deleting all the tiles will still make Unity very lagging.
For the problem of a large number of face fliped in the model, I found a tutorial about double-sided rendering on youtube, and found a double-sided rendering shader in the assets store. If I use a double-sided rendering shader, I don’t have to worry about the problem of face inversion.
I have 6 finished models so far, including Nanchan Temple, Tōdai-ji, Kinkaku-ji, Kōfuku-ji, Chichen Itza, Tutankhamun. I will continue to spend time finding and editing models, but at the same time I am also going to use 6 finished models and start making the first version of the AR app.

Plan for next week

  • Make the first version of the app.
  • Edit the models.
  • Looking for more landmark architecture 3D models.
